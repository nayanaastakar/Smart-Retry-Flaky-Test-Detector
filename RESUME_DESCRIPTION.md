# Resume Description: Smart Retry & Flaky Test Detector

## Project Title
**Smart Retry & Flaky Test Detector** - Intelligent Selenium-Based Automation Testing Framework

## ATS-Friendly Resume Description

### Project Summary
Designed and developed an intelligent test automation framework that automatically retries failed tests, detects flaky tests using AI-powered analysis, distinguishes them from genuine application failures, and generates comprehensive HTML reports. Implemented using Python 3.12+, Selenium WebDriver, PyTest, and integrated OpenAI GPT-4 and Ollama Llama3 for intelligent failure analysis.

- Created Page Object Model architecture for maintainable and reusable test code
- Implemented comprehensive logging system with timestamped entries and file/console output
- Developed automatic screenshot and browser console log capture on test failures
- Built professional HTML report generator with dashboard metrics, charts, and visualizations
- Wrote unit tests for framework components ensuring code quality and preventing regressions
- Applied SOLID principles, design patterns (Strategy, Singleton, Factory, Template Method), and PEP 8 standards


## Architecture
The framework follows a layered architecture:
- **User Interface Layer**: Command-line and PyTest integration
- **Framework Layer**: Core modules (retry, flaky detection, logging, etc.)
- **Page Object Layer**: Page abstractions for test interactions
- **Browser Layer**: Selenium WebDriver for browser automation
